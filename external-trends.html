    Pipeline

    수집 → 정제 → 페이지 노출 흐름

    • 1. 수집
      npm run life:input:external · public RSS/Atom metadata only
    • 2. 분석
      npm run life:loop:external-triage · 우선순위/행동 후보 생성
    • 3. 노출
      external-trends.html · Daily Brief · Agent Map v10 · Live JSON
    • 4. 검증
      npm run deploy:briefing:check · live/static fallback과 페이지 marker 확인
    원문 전체 기사나 유료 데이터, 개인 메시지, secret 값은 포함하지 않습니다. 이 페이지는 public-safe metadata와 요약 신호만 보여줍니다.
